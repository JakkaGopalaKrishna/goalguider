# GoalGuider – Personalized Career Roadmap & Streak System

GoalGuider is a full-stack web application that helps students generate
a personalized career roadmap based on their current status and career goals.
It also includes a streak system to encourage consistent task completion.

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Python, Django, Django REST Framework
- Database: PostgreSQL

## Project Structure
- backend/  → Django backend
- frontend/ → HTML/CSS/JS frontend
